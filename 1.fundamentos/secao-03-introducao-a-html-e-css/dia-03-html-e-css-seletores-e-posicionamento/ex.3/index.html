Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web

Principais motivos para eu gostar deles

  • TikTok-para muitos um aplicativo de dancinha, mas o algoritmo de entrega de conteúdo, faz com que ele se torne muito mais que isso. Basicamente o que a era a MTV nos anos 80 e 90, o TikTok é para a nova geração. Além de ser o start para novas informações importantes. Fazendo o papel de zines e revistas independentes.

  • Pitchfork Media- é umas dos melhores locais pra encontrar música nova boa na internet.

  • Pexels- meu banco de imagens gratuito preferido.
